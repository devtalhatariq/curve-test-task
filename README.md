# How to setup and run

- Run: `bundle install`

- Run: `ruby game.rb`

# Pre-Requirements

- Ruby 2.7.1


# How to run test cases

- Run: `rspec`

# Commands

Following are valid commands

    PLACE 2,2,NORTH
    MOVE
    LEFT
    RIGHT
    REPORT

# Sample:


	PLACE 0,0,NORTH
    LEFT
    REPORT


	Output: 0,0,WEST
